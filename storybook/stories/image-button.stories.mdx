# Image Button

Styled button component which takes an `icon` prop to display an SVG image. Extendable using `styled-components` extension e.g:

```
import { ImageButton } from "@rawpixel-public/react-components";

const RedButton = styled(ImageButton)`
  background-color: red;
`;
```
