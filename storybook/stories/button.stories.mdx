# Button

Styled button component. Extendable using `styled-components` extension e.g:

```
import Button from "@rawpixel-public/react-components/dist/button";

const RedButton = styled(Button)`
  color: red;
`;
```

Use `as` prop to render different elements (see examples);

## Button

A regular `<button>` component with an `onClick` prop.

The `disabled` prop can be set to `true` to disable the button (e.g. for loading states etc). If no `onClick` or `href` is supplied, the button will have the disabled attribute.

<Preview>
  <Story name="button">
    <Button active={boolean("active", false, "button")} onClick={action("button-click")} disabled={boolean("disabled", false, "button")} loading={boolean("loading", false, "button")} size={select("size", ["xsmall", "small", "medium", "large", "xlarge"], "medium", "button")} style={{width: "200px"}}>{text("children", "Click me!", "button")}</Button>
  </Story>
</Preview>

## Link

Render `as` an `a` tag and add an `href` prop.

Additional HTML attributes can be used as per a regular anchor tag e.g. `target=_blank`, `rel=noopener` etc.

<Preview>
  <Story name="link">
    <Button as="a" href="https://www.rawpixel.com" target={text("target", "_blank", "link")} style={{width: "200px"}}>{text("children", "Link elsewhere", "link")}</Button>
  </Story>
</Preview>

## Input

Render `as` and `input` tag and add a `type="submit"` prop. Note that the button text must be supplied as the `value` prop, as React does not allow this element to use `children`.

<Preview>
  <Story name="submit">
    <Button as="input" type="submit" onClick={action("submit-click")} style={{width: "200px"}} value={text("value", "Submit button", "input")} />
  </Story>
</Preview>

## Text field

Render `as` and `input` tag and add a `type="text"` prop. Note that the button text must be supplied as the `value` prop, as React does not allow this element to use `children`. `onChange` should be used instead of `onClick`.

<Preview>
  <Story name="textfield">
    <Button as="input" type="text" placeholder={text("placeholder", "Type in here", "textfield")} onChange={action("on-text-input-change")} style={{width: "200px"}} />
  </Story>
</Preview>
